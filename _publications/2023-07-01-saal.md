Active learning aims to efficiently select the most informative data points for labeling. This work proposes Sharpness-Aware Active Learning (SAAL), which uses maximally perturbed loss as an acquisition score, motivated by generalization bounds.

**Key Contributions:**
- Derived a generalization bound connecting active learning to loss landscape sharpness
- Proposed using maximally perturbed loss as the acquisition function for selecting informative samples
- Demonstrated that flat minima-based calculations lead to more efficient image-label pair acquisition
- Achieved state-of-the-art performance across multiple active learning benchmarks
